INSTRUKSI PENGGUNAAN GAMBAR SUNGAI CISADANE

1. Simpan gambar Sungai Cisadane yang telah dikirimkan ke dalam direktori ini dengan nama 'sungai_cisadane.png'
2. Pastikan gambar disimpan dengan format PNG
3. Gambar akan digunakan sebagai background pada aplikasi SIMOFLOOD
4. Jika gambar tidak tersedia, aplikasi akan menampilkan gradient warna sebagai pengganti

Catatan: Gambar Jembatan Cisadane ini akan memberikan tampilan visual yang lebih menarik pada aplikasi monitoring banjir.
